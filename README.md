# Touch Sensitive Alarm for Emergency Assistance

## Overview

The Touch Sensitive Alarm for Emergency Assistance is an assistive electronic system designed to help wheelchair-bound and mobility-impaired individuals call for help with minimal physical effort. The system activates an audible and visual alarm when the user lightly touches a metal plate, enabling quick emergency communication.

## Objectives

- Design a simple and reliable touch-sensitive alarm system.
- Assist wheelchair users during emergency situations.
- Provide a low-cost and user-friendly solution.
- Ensure fast response with minimal touch input.

## Working Principle

The system detects the weak AC signal naturally present in the human body when a user touches a metallic touch plate. This signal is amplified using an IC 741 operational amplifier and processed by a CA3140 comparator. When the signal exceeds a preset threshold, a BC547 transistor activates a buzzer and LED to generate an emergency alert.

## Components Used

- IC 741 Operational Amplifier
- CA3140 MOSFET Operational Amplifier
- BC547 NPN Transistor
- Resistors and Capacitors
- 12V Piezo Buzzer
- LED Indicator
- Metallic Touch Plate
- 12V DC Power Supply

## Features

- High touch sensitivity
- Fast response time
- Low power consumption
- Adjustable sensitivity control
- Audible and visual alert indication
- Cost-effective and easy to implement
- Suitable for wheelchair integration

## Applications

- Wheelchair emergency assistance systems
- Elderly care support

## Results

The implemented system successfully:

- Detected even very light touch inputs.
- Produced reliable alarm activation without false triggering.
- Achieved fast response with low power consumption.
- Demonstrated suitability for wheelchair-mounted emergency assistance.
